## Sobre o Projeto

**InvestSmart** é uma plataforma de análise de investimentos que combina dados de câmbio em tempo real, preço do Bitcoin e indicadores econômicos para gerar recomendações de investimento personalizadas.

### 🎯 Objetivo

Fornecer aos investidores uma ferramenta simples e acessível para:
- Acompanhar cotações de moedas em tempo real
- Visualizar tendências do mercado
- Obter recomendações baseadas em dados reais

### 🚀 Tecnologias

| Camada | Tecnologias |
|--------|-------------|
| **Backend** | FastAPI, Uvicorn, Pydantic |
| **Frontend** | Streamlit, Plotly, Pandas |
| **Database** | SQLite (dev) / PostgreSQL (prod) |
| **Infra** | Docker, Koyeb, Streamlit Cloud |

### 📊 Funcionalidades

- ✅ Taxas de câmbio de 170+ moedas (Fixer.io)
- ✅ Preço do Bitcoin em tempo real (CoinGecko)
- ✅ Dashboard interativo com gráficos
- ✅ Recomendações de investimento
- ✅ Sistema de autenticação por chave temporária
- ✅ Painel administrativo com estatísticas
